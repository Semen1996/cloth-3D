# Симуляция ткани на основе Position-Based Dynamics (PBD)

Интерактивная симуляция ткани, реализованная на JavaScript с использованием Three.js и метода Position-Based Dynamics (PBD).

Сайт: https://semen1996.github.io/cloth-3D/

## Features

- Симуляция ткани на основе сетки частиц.
- Position-Based Dynamics (PBD) решатель.
- Фиксированные угловые вершины.
- Гармоническое возбуждение центральной точки.
- Рендеринг с помощью Three.js
- Включение и отключение гравитации.
- Работа в реальном времени в браузере.

---

## Управление

### Мышь

- Левая кнопка — вращение камеры.
- Правая кнопка — перемещение камеры.
- Колесо мыши — масштабирование.

### Интерфейс

- **Gravity ON/OFF** — включение и отключение гравитации.

---

## Основные параметры

```js
Nx = 25;
Ny = 25;

dt = 0.001;

iterations = 10;

A = 0.15;
omega = 8.0;
```

| Параметр   | Описание                                 |
| ---------- | ---------------------------------------- |
| Nx, Ny     | Размер сетки ткани                       |
| dt         | Шаг интегрирования                       |
| iterations | Количество итераций решателя ограничений |
| A          | Амплитуда колебаний                      |
| omega      | Частота возбуждения                      |

## Научная основа

Проект основан на работе:

**Müller M., Heidelberger B., Hennix M., Ratcliff J.**
_Position Based Dynamics_, 2007.

Данный подход широко применяется в физических движках и системах моделирования тканей благодаря своей устойчивости и высокой производительности.
